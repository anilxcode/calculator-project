# Basic Calculator in C++

A simple calculator built in C++ that performs basic operations: addition, subtraction, multiplication, and division.

## 👨‍💻 How to Run
1. Compile: g++ calculator.cpp -o calculator
2. Run: ./calculator

## 📋 Sample Input

Enter operator (+, -, *, /): +
Enter two numbers: 5 6
Result: 11



## 🛠 Tech Stack

- *Language:* C++
- *Compiler:* GCC
- *Platform:* Terminal / Command Line



## ✨ Features

- 🖥 Simple terminal-based interface  
- 👶 Beginner-friendly code
